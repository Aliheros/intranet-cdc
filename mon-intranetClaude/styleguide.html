19 Règles d'uniformisation — à appliquer partout
1.
border-radius : 8px formulaires · 10px cards · 12px panels/sections · 16px modales · 20px chips/badges ronds · 99px pills/progress
2.
Padding intérieur cards : 19px (.kc) · 18px (.sc) · 14px (bg-hover-card) · 12px (bg-alt-card)
3.
Titres de section : utiliser .sec (avec ligne automatique) ou .section-label (10px CAPS) — jamais de h3/h4 nus
4.
Boutons d'action inline : 32×32px, border-radius 7, bg bg-alt, border 1px border-light — jamais de texte visible, seulement title tooltip
5.
Couleurs de texte : text-base pour les données importantes · text-dim pour le secondaire · text-muted pour les labels/meta · jamais de #000 ou #333 direct
6.
Espacement vertical : gap 4 (items très proches) · gap 6 (items proches) · gap 8 (standard) · gap 12 (groupes) · gap 20 (sections) · gap 32 (blocs majeurs)
7.
Badges de statut : toujours via <StatusBadge> ou les classes .badge-base + .badge-{color} — jamais de style inline ad hoc pour les statuts
8.
Ombres : 0 10px 15px -10px rgba(15,23,42,0.05) (--shadow-subtle, hover de cards) · 0 24px 80px rgba(0,0,0,0.3) (modales) · pas d'autres valeurs
9.
Transitions : all 0.15s ease (rapide, boutons icônes) · all 0.2s ease (standard) · all 0.3s ease (cards hover, modales)
10.
Focus states : border-color: #1a56db — aucun outline natif (outline: none sur .form-input / .form-select)
